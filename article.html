6 Tipe Data Pemrograman yang Harus Anda Pahami Apa Saja?

Veraa Amanda
17 June 2023
Referance: https://it.telkomuniversity.ac.id/6-tipe-data-pemrograman-yang-harus-anda-pahami-apa-saja/

Tipe data pemrograman merupakan atribut yang berkaitan dengan data yang akan memberi tahu sistem komputer. Sehingga nantinya bisa menafsirkan nilai dari data tersebut. Secara khusus, tipe data adalah format penyimpanan data. Data bisa dalam bentuk variabel untuk tipe data tertentu. Berikut ini beberapa tipe yang sering digunakan untuk pemrograman, antara lain;

Integer

Sering juga ditulis dengan int, integer merupakan tipe data dalam bentuk bilangan bulat. Umumnya data ini digunakan untuk menyimpan angka tanpa pecahan. Tipe data ini tidak memiliki komponen pecahan.

Floating Point

Jika tipe data bilangan bulat masuk dalam integer, maka bilangan pecahan masuk dalam tipe data floating point. Tipe data ini mengacu pada penggunaan dua kali lipat dari penyimpanan relatif. Metode angka yang digunakan sama seperti yang ada di kalkulator.

Character

Tipe data ini sering disebut chat yang digunakan untuk menyimpan satu huruf, angka, tanda baca, simbol atau spasi kosong. Jumlah yang akan Anda simpan dalam tipe data ini cukup banyak. Oleh karena itu penggunaan bitnya harus mewakili setiap karakter yang unik.

Boolean

Sedangkan untuk tipe Boolean merupakan tipe data yang mewakili nilai benar dan salah. Umumnya, tipe data ini dipresentasikan dan hanya mewakili 2 nilai. Hal tersebut mempertimbangkan hubungan angka kecil dan besar.

Array

Tipe data Array berbentuk daftar yang akan menyimpan sejumlah elemen. Namun, memiliki urutan tertentu pada data yang sama. Program yang menggunakan tipe data ini adalah aplikasi pengukur kalori olahraga.

String

Tipe terakhir data adalah String. Tipe data ini paling populer dan paling banyak digunakan. Tipe data bisa mencakup angka, huruf dan karakter lainnya. Pada dasarnya ada banyak karakter berbeda, namun dianggap sebagai teks.

Pemahaman tipe data pemrograman tidak hanya dibutuhkan oleh programmer saja. Orang awam pun bisa mempelajarinya.

10 Istilah dalam Dunia Coding yang Perlu Kamu Tah

Veraa Amanda
25 June 2023
Referance: https://www.gamelab.id/news/151-istilah-istilah-dalam-dunia-per-codingan-yang-wajib-diketahui-pemula

Jika kamu ingin mulai mempelajari coding atau pemrograman, beberapa istilah harus kamu ketahui terlebih dahulu, untuk mempermudah saat sudah melakukan coding nantinya. Dari beberapa istilah ini mungkin sebgaian sudah tidak asing lagi bagi kalian yang sudah sering menggunakan internet atau sering berselancar di mesin pencari, untuk mempersingkat waktu mari kita mulai dari istilah yang pertama.

Algoritma

Algoritma merupakan sebuah kumpulan perintah atau langkah-langkah yang di gunakan oleh programer untuk menjalankan sebuah perintah. Semakin efisien algoritma yang dibuat oleh seorang programmer maka semakin cepat program yang akan dijalankan.

API (Aplication Programming Interface)

API atau Application Programming Interface adalah sebuah sistem atau protokol dan tool yang membanungun aplikasi perangkat lunak yang berjalan di balik layar. API bertugas untuk menyampaikan perintah dari pengguna kepada sistem atau program untuk memberitahukan respon apa yang akan dilakukan atas perintah tersebut. Cara kerjanya yaitu dengan berperan sebagai kontrak yang menetapkan bagaimana sebuah program meminta program lain untuk layanan yang lebih detil.

BUG

Bug bisa disebut sebagai sebuah program cacat atau kesalahan dalam kode, sehingga menyebabkan tampilan, grafis, karakter, sistem dan lainnya membuat sebuah website atau program tidak bekerja seperti seharusnya. Dengan adanya bug dapat menimbulkan celah bagi para hacker untuk memanipulasi atau sengaja menjebol keamanan sistem tersebut yang dapat merugikan developer.

CSS

CSS atau Cascading Style Sheets merupakan salah satu bahasa desain web yang mengontrol format atau menentukan style seperti mengontrol jenis font dan warna pada sebuah halaman web yang ditulis dengan menggunakan sebuah bahasa penanda khusus (markup language).

HTML

HTML adalah singkatan dari Hypertext Markup Language. Disebut hypertext karena di dalam HTML sebuah text biasa dapat berfungsi lain, kita dapat membuatnya menjadi link yang dapat berpindah dari satu halaman ke halaman lainnya hanya dengan meng-klik text tersebut.

Javascript

JavaScript merupakan suatu bahasa pemrograman yang bersifat Client Side Programming Language yang dapat berjalan pada semua web browser. Pengertian dari Client Side Programming Language sendiri merupakan tipe bahasa pemrograman, dan prosesnya dilakukan oleh client. JavaScript banyak digunakan untuk membuat sebuah halaman pada website agar menjadi lebih menarik dan lebih cepat di dalam browser.

jQuery

jQuery adalah kumpulan fungsi-fungsi JavaScript yang memudahkan penulisan kode JavaScript atau sebuah library JavaScript. Dalam dunia coding, library merupakan sebuah kumpulan fungsi yang dapat digunakan untuk memudahkan pembuatan sebuah aplikasi.

Web Server

Secara umum Web Server merupakan sebuah software yang memberikan layanan khusus berbasis data dan berfungsi menerima permintaan dari HTTP atau HTTPS pada klien yang kita kenal dengan nama web browser (Mozilla Firefox, Google Chrome) dan untuk mengirimkan kembali yang hasilnya dalam bentuk beberapa halaman web dan pada umumnya akan berbentuk dokumen HTML.

UI (User Interface)

UI atau User Interface merupakan sebuah bentuk tampilan grafis dibuat sedemikian rupa yang berhubungan langsung dengan pengguna dan berfungsi sebagai penghubung antara pengguna dengan sistem operasi sehingga aplikasi atau program dapat beroperasi sebagaimana mestinya.

UX (User Experience)

User Experience merupakan sebuah proses meningkatkan kepuasan pengguna (pengguna aplikasi, pengunjung website) dalam meningkatkan kegunaan dan kesenangan yang diberikan dalam interaksi antara pengguna dan produk. Secara umum, User Experience merupakan suatu proses membuat sebuah website atau aplikasi menjadi lebih mudah untuk digunakan dan tidak membingungkan ketika digunakan oleh pengguna atau biasa disebut dengan user friendly.

Langkah Awal Untuk Memulai Belajar Pemrograman Web

Veraa Amanda
5 July 2023
Referance: https://www.dicoding.com/blog/memulai-belajar-pemrograman-web/

Menjadi orang awam dalam bidang web bukanlah hal yang memalukan jika ada keinginan belajar dan berkembang. Ada banyak langkah awal untuk memulai belajar pemrograman web mulai dari belajar otodidak, kuliah, kursus, atau belajar online di website yang berbayar ataupun gratis. Tuntutan industri yang serba online membuat kita untuk berlomba lomba belajar bahasa pemrograman terutama web sehingga banyak yang terjun langsung ke dalam jurang coding dengan memimpikan indahnya mempunyai startup besar seperti Tokopedia dan GoJek yang levelnya unicorn. Bahkan dengan mudahnya dapat memimpikan seorang investor yang siap bergabung dengan startup yang kita buat. Tapi pernahkah kalian berfikir mulai dari mana harus belajar? Oke kita akan bahas dalam artikel ini, ada beberapa langkah awal untuk memulai belajar pemrograman web.

Paham Kata "Pemrograman"

Untuk mulai terjun ke dunia pemrograman, mulailah belajar dari yang dasar terlebih dahulu. Karena apabila langsung loncat ke dalam bab yang dalam ditakutkan akan berhenti di tengah jalan. Hal tersebut bisa saja terjadi ketika menemui permasalahan yang sulit tapi karena belum mengereti dasarnya sehingga takut untuk meneruskan belajar. Mulailah dari hal hal kecil, seperti membaca buku sejarah tentang coding atau membaca buku programming yang dikhususkan untuk pemula. Sehingga ada bayangan yang kita tangkap dalam otak, apa itu ngoding.

Belajar Salah Satu Bahasa

Setelah paham dengan kata pemrograman, kita semakin tau bahwasanya bahasa coding itu sangat banyak tapi cara penerapannya sama. Hanya saja setiap bahasa coding ada kekurangan dan kelebihan dalam pembuatan aplikasi ataupun web. Sehingga kita sebagai developer yang baru menyelam harus paham betul apa yang dibutuhkan dalam dunia industri saat ini. Boleh saja mempelajari semuanya, tapi yang paling utama kuasai satu bahasa.

Temukan Bahan yang Dibutuhkan

Ada banyak bahan-bahan yang dibutuhkan sebelum membuat web dan itu bisa dipilih sesuai keinginan, seperti HTML, CSS, PHP, MSQL, JavaScript, dsb yang harus dipelajari. Jika dijelaskan semua pasti tak ada habisnya. Berarti masih banyak yang harus dipelajari, dong? Ya, benar! Kembali ke niat dan tujuan awal yang sudah tertancap di dalam hati.

Buatlah Program Sederhana

Bahan-bahan sudah siap nih, wahh saatnya mulai menyapa dunia dengan barisan kodingmu “Hello Budi, Hello Intan, Hello Sinta” dan sebagainya. Bingung mau buat apa? Buatlah profilmu di dalam barisan kode yang bisa ditampilkan. Buatlah semenarik mungkin hingga menjadi profil yang enak dipandang.

Jalinlah Networking

Menjadi programmer, laptop pasti akan selalu menjadi teman setia. Ya begitulah, udah jomblo halunya tambah gak karuan. Hellow… jika merasa masih banyak yang belum dimengerti jangan semuanya dilakukan sendiri, masalah dipendam sendiri, atau pun error dihadapi sendiri. Mulailah berjejaring seperti lewat komunitas atau forum social media yang sudah terkenal, serta seringlah bertanya di situs-situs pemrograman yang siap membantumu mencari jalan keluar apabila terdapat error. Sehingga kita dapat menyelesaikan permasalahan yang sedang terjadi dan mengembangkan pengetahuan dengan informasi terbaru.
